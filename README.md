# Application de Gestion Multi-Boutiques

## Identifiants par défaut
- Super Admin : superadmin / SuperAdmin123!
- Admin Dakar : admin_dakar / admin123
- Admin Thiès : admin_thies / admin123

## Fonctionnalités
- Multi-boutiques avec isolation des données
- Gestion des produits, stocks, clients
- Caisse (POS) avec panier
- Commandes et factures PDF
- Rapports et statistiques
- Interface d'administration des boutiques

## Installation
1. Cloner le dépôt
2. Créer un environnement virtuel
3. pip install -r requirements.txt
4. python run.py
